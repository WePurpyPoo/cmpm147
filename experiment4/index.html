Experiment 3 - Alternate Worlds

Help from thkaspar's tileset, ChatGPT, and Wes's code

Description

In this assignment, we are creating tile based generative worlds. We must make 1 outdoor world and 1 dungeon world with signs of life being implemented in both.

Technical

The system runs on many functions but mainly, a grid is created using symbols such as '_' and ','. Based on which symbol is on the grid and what it is surrounded by, the program shows a small image that is from a tile based image. It does this by using a coordinate system for the image to decide what it shows.
